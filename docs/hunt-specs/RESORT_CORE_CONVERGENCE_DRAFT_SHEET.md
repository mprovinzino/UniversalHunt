# Resort Core Convergence Draft Sheet

Working authoring sheet for the resort-wide championship hunt.

This file is intentionally broader than the current live challenge library.
It can use:

- existing app anchors
- anchors already named in the hunt specs
- new proposed anchors that still need to be field-validated and added later

## Mainline Rules

Mainline steps should prefer:

- fixed landmarks
- obvious facades
- stable signage
- vista points
- repeatable photo proof

Mainline steps should avoid:

- cast or character interaction
- staff-dependent questions
- scheduled live encounters
- variable dialogue
- anything that becomes impossible when one performer, set, or show schedule changes

Allowed only as bonus or limited-time variants:

- Gringotts Money Exchange goblin
- Shrek's Swamp Meet
- Raptor Encounter
- show-dependent prompts that require a performance window
